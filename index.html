Material Smart Banner

Promote your Native Android App on the mobile web, with a few lines of Javascript.

Get Started

As an external script


<meta name="material-smart-banner" content="name=Photo App, url=http://google.com/play, icon=https://unsplash.it/80/80?image=301">
<script src="http://npmcdn.org/materialSmartBanner/dist/materialSmartBanner.min.js"><script>

As an NPM script (npm i materialsmartbanner --save)


import materialSmartBanner from 'materialSmartBanner';

document.addEventListener("DOMContentLoaded", () => {
  const options = {
    name: 'Photos.io App',
    url: 'http://google.com/play/myApp',
    icon: '/images/myAndroidIcon.jpg'
  };
  new materialSmartBanner(options);  
})

Available Options

<meta> options, or new materialSmartBar({options})

Option Default
name APP NAME NOT FOUND
closeIcon ⨯
subText GET — On the Play Store
linkText View
linkUrl http://google.com/play
icon https://unsplash.it/80/80?image=404

Why Does this exist?

iOS Safari/Chrome gives us smart banners to promote our native iOS apps, but there isn't a default like this for Android Apps.
